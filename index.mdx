## Try in your browser

Most demos are exported to GitHub Pages and can be [played directly in your browser](https://godotengine.github.io/godot-demo-projects/) without installing anything.

<Note>
  Browser performance is lower than native desktop. For best results, download and run demos locally in the Godot Editor.
</Note>

## Getting started

<Steps>
  <Step title="Install Godot Engine">
    Download the latest stable release from [godotengine.org](https://godotengine.org). For C# demos, use the **.NET** version.
  </Step>
  <Step title="Clone or download the demos">
    Clone this repository or download a ZIP archive from GitHub and extract it to a folder.
  </Step>
  <Step title="Scan the folder in the Project Manager">
    Open Godot's Project Manager, click **Scan**, and point it at the folder containing all demos. All projects will appear in the list.
  </Step>
  <Step title="Open and run any demo">
    Double-click any project to open it, then press **F5** (or click **Run**) to play it.
  </Step>
</Steps>
